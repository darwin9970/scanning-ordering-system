## 五、新增工具和组件

### 5.1 工具类

#### Toast 提示工具 (`mini/utils/toast.js`)
- `showSuccess(title, duration)` - 成功提示
- `showError(title, duration)` - 错误提示
- `showWarning(title, duration)` - 警告提示
- `showInfo(title, duration)` - 信息提示
- `showActionSuccess(action, detail)` - 操作成功提示
- `showActionError(action, reason)` - 操作失败提示
- `showLoading(title)` - 加载提示
- `hideLoading()` - 隐藏加载提示

#### 错误处理工具 (`mini/utils/error-handler.js`)
- `handleApiError(error, options)` - 处理 API 错误
- `showNetworkError(action, onRetry, showRetry)` - 显示网络错误
- `handleHttpError(status, action)` - 处理 HTTP 状态码错误
- `showFormError(field, message)` - 显示表单验证错误

#### 页面过渡工具 (`mini/utils/page-transition.js`)
- `pageEnter(type)` - 页面进入动画配置
- `pageExit(type)` - 页面退出动画配置
- `animateNavigation(options)` - 导航栏动画配置

#### 订单状态通知工具 (`mini/utils/notifications.js`)
- `notifyOrderStatusChange(order)` - 订单状态变更通知
- `startOrderStatusPolling(orderId, onStatusChange)` - 订单状态轮询
- `listenOrderStatus(orderId, callback)` - 监听订单状态（WebSocket）

### 5.2 组件

#### 会员卡片组件 (`mini/components/member-card/member-card.vue`)
- 突出显示会员等级、积分、优惠券数量

#### 骨架屏组件 (`mini/components/q-skeleton/q-skeleton.vue`)
- 支持多种类型：list、card、text、avatar
- 支持动画效果
- 可配置行数、尺寸等

#### 积分抵扣组件 (`mini/components/points-deduction/points-deduction.vue`)
- 显示可用积分
- 实时计算抵扣金额
- 支持开关控制

#### 滑动操作组件 (`mini/components/q-swipe-action/q-swipe-action.vue`)
- 支持左右滑动
- 可配置多个操作按钮
- 支持不同操作类型（danger、primary、warning）
