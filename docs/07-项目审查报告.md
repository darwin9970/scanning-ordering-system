## 📊 项目概览

### 技术架构
- ✅ **后台管理**: Next.js 16 + TypeScript + TailwindCSS + shadcn/ui
- ✅ **API服务**: Elysia + Drizzle ORM + PostgreSQL + Redis
- ✅ **小程序**: uni-app + Vue 3 + Pinia
- ✅ **代码质量**: ESLint + Prettier + Husky + lint-staged

### 核心功能
- ✅ 完整的商家后台管理系统
- ✅ 页面装修系统（自由画布）
- ✅ 订单管理（加菜、退款）
- ✅ 商品管理（多规格、属性）
- ✅ 角色权限（RBAC）
- ✅ 优惠券、会员、营销活动
- ✅ 实时通知（WebSocket）
