# 第二部分：各页面详细分析

## 一、首页 (tabIndex=0)

### 1.1 子Tab详细配置

#### 子Tab 1: 首页
主首页内容配置，完整的页面编辑器

#### 子Tab 2: 加载页
小程序启动加载页配置（Splash Screen）
- **功能开关**: 开关控制是否显示加载页
- **操作按钮**: 保存

##### 加载页内容配置
| 配置项 | 类型 | 说明 |
|-------|-----|-----|
| 上传类型 | 单选 | 图片/视频 |
| 图片/视频 | 上传 | 建议尺寸750*1630，大小不超过2M |
| 动画效果 | 下拉选择 | 淡出等效果 |
| 静止时长 | 数字 | 2-10秒 |

##### 跳过按钮配置
| 配置项 | 类型 | 说明 |
|-------|-----|-----|
| 按钮文本 | 文本 | 如"跳过" |
| 文本样式-字号 | 数字 | 10-16 |
| 文本样式-颜色 | 颜色 | rgba格式 |
| 边框-粗细 | 数字 | 0-5 |
| 边框-颜色 | 颜色 | rgba格式 |
| 填充-颜色 | 颜色 | rgba格式（按钮背景色） |
| 圆角 | 滑块 | 0-100 |

## 二、点餐页 (tabIndex=1)

### 2.1 子Tab详细配置

#### 子Tab 1: 点餐页
主点餐页面布局配置，与首页编辑器类似

#### 子Tab 2: 优惠券弹窗
进入点餐时的优惠券领取弹窗样式
- **操作按钮**: 保存
- **预览区展示**: 优惠活动弹窗，展示可领取的优惠券列表
  - 买一送一券
  - 满减券
  - 新人礼包
- **配置项**: 弹窗样式预设，暂无自定义配置

#### 子Tab 3: 选择人数
就餐人数选择弹窗样式
- **操作按钮**: 保存、恢复默认、保存到记录
- **可用组件**:
  - 文本
  - 图片
  - 桌号
  - 门店名称
  - 门店头像
- **预览区展示**: 人数选择弹窗（1-9人+其他）
- **配置项**: 背景图、按钮样式

#### 子Tab 4: 门店地址提示
门店位置确认弹窗样式
- **操作按钮**: 保存
- **配置项**:
  | 配置项 | 类型 | 说明 |
  |-------|-----|-----|
  | 地址提示 | 开关 | 是否开启地址提示 |
  | 提示样式 | 选择 | 两种预设样式可选 |
- **预览区展示**: 附近门店列表弹窗

#### 子Tab 5: 就餐方式选择
堂食/外卖/自取选择弹窗样式
- **操作按钮**: 保存、恢复默认
- **配置项**:
  | 配置项 | 类型 | 说明 |
  |-------|-----|-----|
  | 自定义设计 | 开关 | 是否开启自定义设计 |
  | 功能开关 | 选择 | 开启/关闭，关闭后不再提示选择 |
  | 样式 | 选择 | 预设样式选择 |
- **就餐方式配置**（堂食/自取/外卖/快递各自配置）:
  | 配置项 | 说明 |
  |-------|-----|
  | 显示开关 | 是否显示该就餐方式 |
  | 标题内容 | 如"店内堂食"、"打包带走" |
  | 未选图标 | 未选中状态图标 |
  | 选中图标 | 选中状态图标 |
- **其他配置**:
  | 配置项 | 说明 |
  |-------|-----|
  | 文本样式 | 字号(10-16)、颜色、加粗 |
  | 圆角 | 0-100，支持三种预设+自定义 |
